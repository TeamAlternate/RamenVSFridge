ゲーム概要
　ラーメンはトッピングを集めて二郎系になることが目的
　冷蔵庫はラーメンを冷やすのが目的
　鬼ごっこです


操作方法
　ラーメン
　　移動します
　冷蔵庫
　　移動します
　　ラーメン冷やしアタックが炸裂します


いろいろ説明
　ラーメンはあったかポイントに入ることで再加熱できます



その他機能たち
　Escキーでゲームを終了します

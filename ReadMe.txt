ゲーム概要
　ラーメンはトッピングを集めて二郎系になることが目的
　冷蔵庫はラーメンを冷やすのが目的
　鬼ごっこです


操作方法
　ラーメン
　　移動します
　　冷蔵庫にアタックします
　冷蔵庫
　　移動します
　　ラーメン冷やしアタックが炸裂します


いろいろ説明
　ラーメンはあったかポイントに入ることで再加熱できます



その他機能たち
　Escキーでゲームを終了します
　


こだわりポイント
　・カメラの動き
　　カメラの移動・ズームを滑らかにして
　　画面酔いしにくくなるように工夫してます
　　実は内部はフレームレートに依存しない処理方法になってます
　・シーン遷移
　　関数呼び出し一回でシーン遷移・アニメーション再生を一括で発動できるようにしてます
